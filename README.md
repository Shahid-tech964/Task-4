<h1>Task-4</h1>

<p>
Little Explorer is a Flutter learning app designed for children.
The app uses the device camera and ML Kit object detection to help children identify real-world objects in an interactive and engaging way.
</p>

<h2>Features</h2>

<ul>
  <li>📷 Camera integration</li>
  <li>🤖 ML Kit object detection</li>
  <li>🎯 Find 5 objects activity</li>
  <li>📊 Progress tracking</li>
  <li>🏆 Reward screen</li>
  <li>🎨 Child-friendly interface</li>
</ul>

<h2>How It Works</h2>

<ol>
  <li>Open the app.</li>
  <li>Point the camera at objects.</li>
  <li>ML Kit detects objects in real time.</li>
  <li>Complete all 5 object challenges.</li>
  <li>Unlock the reward screen.</li>
</ol>

<h2>Technology Used</h2>

<ul>
  <li>Flutter</li>
  <li>Dart</li>
  <li>Google ML Kit Object Detection</li>
  <li>Camera Package</li>
</ul>
